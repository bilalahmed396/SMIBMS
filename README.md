# Social Media Intelligence & Brand Monitoring System (SMIBMS)

## Project Overview
This project is a Sentiment Analysis system designed for airline brands using a **Hybrid BiLSTM-RoBERTa** deep learning architecture. It classifies social media tweets into **Positive**, **Negative**, or **Neutral** categories.

## Week 5 Features
* **Hybrid Model**: Integration of RoBERTa embeddings with BiLSTM for temporal sequence modeling.
* **User Interface**: Real-time sentiment prediction using a Gradio-based web interface.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the UI: `python app.py`
